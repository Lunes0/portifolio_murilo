## 🛠️ Instalação

### Clone o repositório:

- git clone https://github.com/Lunes0/projeto_efood/tree/Parte-4

- cd efood

### Instale as dependências:

- npm install

---

## ▶️ Executando o projeto

### Ambiente de desenvolvimento:

- npm run dev

- Acesse em: http://localhost:5173

### Build de produção:

- npm run build

### Pré-visualização do build:

- npm run preview

## 🧪 Testes

- npm run lint
